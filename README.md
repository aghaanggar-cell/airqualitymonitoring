# Air Quality Monitoring System

Air Quality Monitoring System (AQMS) adalah sistem yang digunakan untuk memantau kualitas udara secara real-time menggunakan berbagai sensor lingkungan.

Sistem ini membantu industri, pemerintah, dan perusahaan manufaktur untuk memastikan kualitas udara tetap sesuai dengan standar lingkungan.

## Fungsi Air Quality Monitoring System

- Monitoring PM2.5 dan PM10
- Monitoring gas CO, SO2, NO2
- Dashboard monitoring real-time
- Sistem alarm kualitas udara
